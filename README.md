# 💻 Clon simple de un foro
Aplicación Fullstack que permite interactuar en un pequeño foro, posteando, reaccionando y respondiendo a otro post o comentarios.

## 🚀 Tecnologías utilizadas
- Frontend: React 19+
- Backend: Node.js
- Estilo: CSS
- Base de datos: MongoDB
- Gestión de versiones: Git / GitHub.

## 🛠 Instalación
Clona el repositorio y arranca los servicios (Windows):

```
git clone https://github.com/PalomaSH2001/ForoChan.git
cd ForoChan
```

Backend (Node.js)

```
cd backend
npm install
npm run dev
```

Frontend (React)

```
cd ../frontend
npm install
npm run dev
```
La aplicación frontend por defecto estará disponible en: http://localhost:5173/
